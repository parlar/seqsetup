Dashboard
=========

After logging in, you land on the dashboard which provides an overview of your
sequencing runs.

Run List
--------

The dashboard displays existing sequencing runs in a table, showing:

- Run name
- Instrument platform
- Flowcell type
- Number of samples
- Creation date
- Current status

From the dashboard you can:

- **Create a new run** -- Starts the run setup wizard
- **Open an existing run** -- Continues editing or reviewing a run
- **Delete a run** -- Removes a run and all its data

Navigation
----------

The top navigation bar provides access to:

- **Dashboard** -- Return to the run list
- **Admin** -- Administrative functions (admin users only)
- **User info** -- Shows the logged-in user with logout option
